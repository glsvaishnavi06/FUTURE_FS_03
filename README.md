# GVRK Bliss Bakery 🍰

A modern and responsive bakery website built using React.js as part of the Future Interns Full Stack Web Development Internship.

## 🌟 Features

* Responsive design for desktop, tablet, and mobile devices
* Attractive bakery landing page
* Product showcase with images
* About Us section
* Services section
* Gallery section
* Contact section
* Smooth navigation and user-friendly interface

## 🛠️ Technologies Used

* React.js
* JavaScript
* HTML5
* CSS3

## 📦 Installation

Clone the repository:

git clone https://github.com/glsvaishnavi06/FUTURE_FS_03.git

Install dependencies:

npm install

Run the project:

npm start

## 🎯 Project Objective

The objective of this project was to develop a professional bakery website with an attractive user interface, responsive layout, and modern web development practices.
