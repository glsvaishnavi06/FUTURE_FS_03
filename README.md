# GVRK Bliss Bakery 🍰

A responsive bakery website built using React.

## Features
- Home, About, Services, Menu, Gallery, Contact sections
- Smooth scrolling navigation
- Image-based product showcase
- Responsive design for mobile and desktop

## Tech Stack
- React JS
- HTML5
- CSS3

## Project Purpose
This project was built as part of a full-stack web development internship task to create a real-world business website.
